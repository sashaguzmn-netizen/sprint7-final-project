# Proyecto Final Sprint 7 - ConnectaTel

## Descripción del proyecto

El objetivo de este proyecto es analizar el comportamiento de los clientes de ConnectaTel a partir de información demográfica y patrones de uso de servicios de telecomunicaciones.

A través del análisis exploratorio de datos, segmentación de clientes y detección de patrones de consumo, se busca identificar oportunidades de negocio y generar recomendaciones para la empresa.

---

## Datasets utilizados

El proyecto utiliza los siguientes datasets:

- plans: información de los planes disponibles.
- users: información demográfica de los clientes.
- usage: registros de uso de llamadas, mensajes y datos.

---

## Etapas del análisis

1. Exploración de los datos.
2. Revisión de estructura y tipos de datos.
3. Limpieza y validación de información.
4. Análisis descriptivo.
5. Visualización de distribuciones.
6. Detección de outliers.
7. Segmentación de clientes por edad y nivel de uso.
8. Generación de insights y recomendaciones de negocio.

---

## Principales hallazgos

- La mayoría de los clientes presentan niveles de uso bajos o moderados.
- Se identificaron usuarios con consumos significativamente superiores al promedio.
- El segmento de adultos representa la mayor parte de la base de clientes.
- Los usuarios de alto consumo representan una oportunidad para desarrollar planes especializados.
